  • Modeling Time needed at tourist attractions

    Co-authours: Elínborg Ásta and Margrét Ásta

    The objective of this project was to use Machine Learning to determine the time needed to spend at a particular tourist attraction in no particular country. By pulling data containing information regarding diversed set of tourists attractions all around the world the program will be more accurate whereas the input is attractions in many countries around the globe. This is a classification problem, where tourist attractions are classified into time intervals that represent the suggested time to spend there.

    Github repository:

    https://github.com/Mario1502/Modeling-time-needed-at-tourist-attractions
